# human-human-control-competition-collaboration
Modeling code for the paper "Representation of a partner’s goals and future actions tunes intelligent reflexes during collaboration and competition"

# Instructions
## 1. Using virtual environment
  - To use the exact same virtual environment, install uv on your system 
  - In the terminal, navigate to the directory where you cloned this repository and run 
  ```bash
  uv sync
  ```
  - NOTE: If using other virtual environments, you can install dependencies individually by looking at pyproject.toml
      - You may need to run
      ``` bash
      pip install -e .
      ```
      to locally install src as a package
